# diving-with-seals
anonymized codebase and web application submitted in conjuction with paper submission. 

## Controls
### 2D devices
Mobile: pinch to zoom, drag to orbit.
Desktop: Click+drag to orbit, Shift+click+drag to pan. Clicking on a seal pathway triggers a callout and view of the multimodal graphs.
### WebXR
Using the right controller, press the grip button and pull towards oneself to move. Use raycast selection by pointing and pulling a trigger to set a callout. The joystick can be used to nudge the callout and animate the seal position.

## Local Development
`npm install` and `npm run dev`
